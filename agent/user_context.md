# Thông tin người dùng & Ngữ cảnh dự án

- **Nền tảng gốc:** Kỹ sư BIM.
- **Lĩnh vực của dự án này:** Tạm gác BIM sang một bên, dự án `E_CYBER-FINANCIAL` sẽ tập trung và đi sâu hoàn toàn vào mảng **Tài chính (Finance)**.
- **Mục đích code:** Các đoạn code Python phục vụ cho công việc thực tế, phân tích/tự động hóa dữ liệu trong lĩnh vực Tài chính.
- **Kinh nghiệm làm việc với AI:** Đã làm việc với Antigravity được một thời gian qua 2 dự án tool tự động hóa cho Revit và Navisworks (có trên GitHub).
- **Mô hình phối hợp (Phân công nhiệm vụ):**
  - **Người dùng (Bạn):** Đóng vai trò là **Kiến trúc sư phần mềm (Software Architect)** — người đưa ra yêu cầu, phân tích và "bơm" logic nghiệp vụ (domain Tài chính), quyết định luồng hoạt động của tool.
  - **AI (Antigravity):** Đóng vai trò là **Lập trình viên (Coder/Programmer)** — người tiếp nhận logic từ bạn để chuyển hóa thành code Python thực tế, lo phần kỹ thuật, thư viện và tối ưu hóa hệ thống.
- **Cách giao tiếp cần thiết:** Đi thẳng vào kết quả thực chiến và giải quyết các bài toán logic của mảng Tài chính. Không lan man các lý thuyết hàn lâm về phần mềm.
